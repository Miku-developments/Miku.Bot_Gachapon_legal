## Gachapon Pinger

Your assistant for **Gachapon card drops** — watches drops in real time, surfaces wishlist data instantly, and helps you stay on top of missions and cooldowns.

**Invite:** https://discord.com/discovery/applications/876139135507763280

---

### ⚡ What it does

**Automatic drop analysis**
- Watches Gachapon drops as they happen and scans each card with image recognition
- Replies with character, series, **Current wishes** (❤️) and **All time high wishes** (💙) wishlist counts
- Supports **1–4 card drops**, including poster / single-card pulls
- Shows **rarity tier** (Legendary → Common) from AGWL so you can decide faster
- Flags **🎯 Mission** cards when they match active weekly or daily goals

**Mission tracking**
- Auto-detects current **weekly** (series) and **daily** (character) missions
- Marks matching cards on every drop
- Tracks progress with `M.mission` — see how many mission cards you’ve pulled today and this week

**Drop reminders**
- Never miss your next pull window — set a personal cooldown reminder with `M.remind`
- Choose **10 min**, **20 min**, or a **custom interval** (1–120 minutes)

**Wishlist intelligence**
- Built-in database covering **thousands of characters and posters**
- Wishlist data stays updated from live Gachapon activity
- See both current and all-time wishlist values at a glance

**Server-wide visibility**
- Keeps collectors informed automatically — no manual lookup during competitive drops
- Global stats via `M.botinfo`: total drops scanned, rarity breakdown, mission hits, uptime, and more

---

### 🧠 Why use it

- **Stop missing high-value cards** — wishlist counts and rarity tiers appear the moment a drop lands  
- **React faster** during busy drop windows when every second counts  
- **Stay on mission** without tabbing out to check weekly/daily goals  
- **Drop on schedule** with reminders so you don’t lose streaks or cooldown timing  
- **Built for Gachapon flow** — not a generic bot; tuned around how these drops actually work

### 🛠 Commands

**Prefix:** `M.`  
Hybrid commands also work as **slash commands** (`/help`, `/remind`, etc.)

| Command | What it does |
|--------|----------------|
| `M.help` | Full command list and setup info |
| `M.remind` | Set drop reminder — 10 / 20 / custom min, or turn off |
| `M.mission` | Weekly & daily mission progress |
| `M.botinfo` | Bot status, uptime, and global drop stats |
| `M.ping` | Latency check |

Card analysis runs **automatically** on Gachapon drops — no command needed.

Use **`M.help`** after inviting to see everything available on your server.
